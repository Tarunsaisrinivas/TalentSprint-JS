### Map:
- Map interface is not a subtype of the collections interace . It has key and value
a. keys should be unique, but values can b duplicated.
b. It allows one null key in implementations like hashmap and linkedHashmap, and allows multiple null values in most implementations.
- Two interfaces :
    - Map interace 
    - Sorted Map Interface
- Three Primary Classes 
    - Hash Map
    - TreeMap
    - LinkedHashMap
